Stock Visualizer

A simple and clean web project that shows real-time stock prices and live line charts for top companies using HTML, CSS, JavaScript, Chart.js, and the AlphaVantage API.

=> Features

1.Live stock prices (auto-updates every 30 seconds)

2.Line chart for each company

3.Company logos included

4.Simple and clean UI

5.Fully frontend (no backend needed)

=> API Key

This project uses a free AlphaVantage API key.
It is safe to keep the key public.
Worst case: API limit gets used up → you can generate a new key.
